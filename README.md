🧠 FIRST LEGO League 2022 – PyBricks Robotics Project

This repository contains the code, documentation, and resources for our group project developed for the FIRST LEGO League Challenge 2022. The project focuses on designing, building, and programming an autonomous LEGO robot using PyBricks, a Python-based firmware for LEGO® SPIKE Prime / EV3 systems.

🚀 Project Overview

Our robot was developed to complete mission-based challenges on the FIRST LEGO League competition table. Using PyBricks, we implemented Python control logic for precise movement, sensor integration, and autonomous decision-making.

🧩 Key Features
Python-based robot control using PyBricks
Sensor integration (color, distance, and touch sensors where applicable)
Autonomous navigation and mission execution
Modular and reusable code structure for different missions
Iterative testing and optimization for competition performance

To design a reliable and efficient autonomous robot capable of completing multiple FIRST LEGO League missions while demonstrating teamwork, engineering design, and programming skills.
